Tout ce que vous devez savoir sur le parrainage Lime

Comment activer le parrainage Lime ?

Il suffit de cliquer sur ce lien de parrainage et de créer un compte Lime. Aucun code à saisir manuellement : les 5 € sont crédités automatiquement dans votre portefeuille de l'application.

Combien offre le parrainage Lime en 2026 ?

Le lien de parrainage Lime offre actuellement 5 € de crédit pour tout nouvel utilisateur s'inscrivant via le lien. Ce montant peut évoluer selon les promotions en cours.

Les anciens utilisateurs peuvent-ils en profiter ?

Non. L'offre de parrainage Lime est exclusivement réservée aux nouveaux utilisateurs qui n'ont jamais créé de compte sur la plateforme. Si vous avez déjà un compte, le crédit ne sera pas appliqué.

Dans quelles villes Lime est-il disponible ?

Lime est présent dans de nombreuses villes françaises : Paris, Lyon, Marseille, Bordeaux, Lille, Nantes, Toulouse et bien d'autres. Vos crédits sont valables dans toutes les villes couvertes par Lime, y compris à l'étranger.

Comment ajouter un code promo manuellement dans Lime ?

Dans l'application Lime, rendez-vous dans l'onglet Portefeuille, puis touchez Ajouter un code promo. Saisissez votre code pour créditer votre solde. Avec le lien de parrainage, cette étape est automatique.

Quels véhicules puis-je utiliser avec mon crédit ?

Les crédits obtenus via le parrainage sont utilisables sur l'ensemble des véhicules Lime disponibles : trottinettes électriques, vélos électriques et scooters électriques, selon les disponibilités locales.

Pourquoi choisir Lime ?

Lime est l'un des leaders mondiaux de la micromobilité partagée. Présent dans plus de 200 villes à travers le monde, Lime propose des trottinettes électriques, des vélos électriques et des scooters électriques en libre-service.

L'application est simple, sécurisée et permet de se déplacer rapidement en ville tout en réduisant son empreinte carbone. Chaque trajet Lime remplace potentiellement un trajet en voiture.
